Структура проєкту
project/
│── main.py                
│── test_go.go              
│── transport_company.sql
│── .env                    
│── README.md              

Вимоги
MySQL Server
Python 3.9 або новіше
Go 1.20 або новіше

База даних
База даних надається у вигляді експортованого SQL-файлу
transport_company.sql.
Для тестування необхідно лише імпортувати цей файл у MySQL
(через MySQL Workbench або командний рядок).
Структура бази даних та тестові дані не повинні змінюватися,
оскільки це вплине на коректність результатів експерименту.

Налаштування підключення
Параметри підключення до бази даних зберігаються у файлі .env,
який вже присутній у проєкті.
Перед запуском програм необхідно відредагувати файл .env
та вказати актуальні параметри вашого локального середовища MySQL
(хост, користувач, пароль тощо).

Запуск програми мовою Python
Встановити залежності:

pip install mysql-connector-python python-dotenv

Запустити програму:

python main.py

Програма виконує фіксований набір SQL-запитів кілька разів та
виводить середній, мінімальний і максимальний час їх виконання.

Запуск програми мовою Go
Встановити залежності:

go mod init db_test
go get github.com/go-sql-driver/mysql
go get github.com/joho/godotenv

Запустити програму:

go run test_go.go

Програма мовою Go виконує той самий набір SQL-запитів над тією ж
базою даних і виводить статистику часу виконання.
